# About Project

This project is a Text-Based Adventure Game where the user makes choices to navigate through a story. It was built to strengthen my understanding of control flow, functions, and basic I/O operations in Python.
* Interactive story with multiple paths.
* Simple text-based user interface.
* Keeps track of player inventory and health.
